# 食用指南

## 一、注册 Cloudinary 账号

1. 访问 [Cloudinary](https://cloudinary.com)
2. 点击 **Sign Up for Free** 注册免费账号
3. 注册完成后会进入控制台，无需付费（免费套餐每月 25GB 流量）

---

## 二、配置 Fetch 白名单

Cloudinary Fetch 需要在控制台添加允许的源站域名，否则无法代理图片。

1. 登录 [Cloudinary Console](https://cloudinary.com/console)
2. 点击左侧 **Settings**（齿轮图标）→ **Security**
3. 滚动到 **Allowed fetch domains** 部分
4. 点击 **Add domain**，添加以下域名：
   ```
   himawari.asia
   ```
5. 点击 **Save** 保存

---

## 三、获取 Cloud Name

1. 登录 Cloudinary 控制台
2. 在 **Dashboard** 首页即可看到 **Cloud Name**（例如 `abc123xyz`）
3. 复制该值备用

---

## 四、在 App 中配置

1. 安装并打开馒头地球
2. 长按壁纸进入 **高级设置**
3. 找到 **Cloudinary CDN** 分类
4. 在 **Cloud Name** 输入框中填入你的 Cloud Name
5. 返回即可，App 会自动通过 CDN 获取卫星图像

---

## 五、架构说明

```
向日葵卫星服务器 (himawari.asia)
         ↓ Cloudinary Fetch 代理
   Cloudinary CDN 缓存
         ↓
      你的设备
```

- **无需 API Key / Secret**：本项目仅使用 Cloudinary Fetch 代理功能，不需要上传，纯读取模式
- **自动图片优化**：CDN 会自动进行格式转换（WebP/AVIF）和压缩，节省流量

---

## 六、常见问题

### Q: 不填 Cloud Name 会怎样？

A: App 无法通过 CDN 获取图像，壁纸将不会更新。

### Q: Cloud Name 填错了会怎样？

A: 图片加载会失败，请检查是否正确复制了控制台首页的 Cloud Name。

### Q: 免费套餐够用吗？

A: 免费套餐每月 25GB 流量。每次更新约消耗几百 KB，正常使用绰绰有余。

### Q: 如何验证 CDN 是否生效？

A: 开启调试模式（高级设置 → 调试模式），壁纸上会显示抓取状态和调试信息。
